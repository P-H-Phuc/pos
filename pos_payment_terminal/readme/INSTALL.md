This module is designed to be installed on the *main Odoo server*. On
the *POSbox/IoTbox*, you should install the module *hw_x* depending on
the protocol implemented in your device.

Ingenico \<http://en.wikipedia.org/wiki/Ingenico\> card readers for
France support the Telium protocol implemented in the
*hw_telium_payment_terminal* module for the POSbox/IoTBox.

You can also consider [pywebdriver
project](https://github.com/akretion/pywebdriver) as an alternative to
the POSbox/IoTbox.
